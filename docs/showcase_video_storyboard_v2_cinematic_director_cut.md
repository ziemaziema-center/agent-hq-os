# Showcase Video Storyboard v2: Cinematic Director Cut

Target length: 75 seconds  
Primary use: Claude Code / OSS grant application, GitHub README media, launch post  
Creative standard: senior technical documentary trailer, not SaaS marketing  
Core promise: AI agents need runtime discipline before they deserve autonomy.

## Executive Creative Brief

This video should feel like watching an experienced operator rebuild order around a chaotic AI-assisted engineering session.

The emotional arc:

1. Cold-start fatigue.
2. Context becomes memory.
3. Memory becomes runtime state.
4. Runtime state meets human gates.
5. Validation turns confidence into evidence.
6. Telemetry turns one session into the next session's advantage.

The final impression should be:

> This maintainer understands the operational failure modes of AI coding agents and is building public scaffolds around them.

## Visual Bible

Use a restrained operator-console world:

- Background: near-black, charcoal, deep navy.
- Accent colors: validation green, warning amber, review blue, stop red used sparingly.
- Typography: monospace for terminal/state, clean sans for titles.
- Camera: slow push, slow pullback, lateral drift, no fast spinning.
- UI style: abstracted GitHub/terminal/runtime panels, never real credentials or real private dashboards.
- Texture: subtle scanlines, faint terminal glow, soft depth of field.
- Motion: cards slide into state, gates lock, telemetry lines flow, PASS appears with restraint.

Avoid:

- robots, androids, glowing brains, sci-fi faces;
- fake code walls full of nonsense;
- claims of full autonomy;
- stock-office footage;
- crypto/trading visuals that look like financial advice;
- real Telegram or n8n account screenshots;
- any readable token, private URL, production ID, or credential.

## Sound Bible

Music:

- Low analog pulse, 70-85 BPM.
- Minimal percussion.
- No heroic trailer drums.
- Rise slightly at validation and final portfolio reveal.

Sound design:

- Soft terminal keystrokes.
- Low gate click for deferred gates.
- One clean validation tone on PASS.
- Subtle paper/card movement for memory files.
- Very light sub-bass on title reveal.

Voice:

- Calm, technical, adult.
- Pace: 125-140 words per minute.
- No hype voice.
- Read like an engineer explaining a hard-earned operating principle.

## Master Narration Script

AI agents can write code. But long-running projects fail when every session starts cold.

Agent HQ OS treats context as runtime state.

It starts with boot memory: known failures, validated patterns, and patch history.

Work moves through queues, reviewer loops, and deferred gates before risky actions can happen.

Validation comes before confidence.

Retries are allowed only when the failure is actually retryable.

Telemetry turns the outcome of one session into the starting point for the next.

This is not autopilot. It is safe bounded autonomy with human review where it matters.

Agent HQ OS is a public scaffold for practical AI runtime operations.
