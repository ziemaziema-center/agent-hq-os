# Workflow Comparison

## One-off Prompting

Fast for a single question. Weak for multi-day execution because memory, validation, and decisions are scattered.

## IDE-only AI

Useful for edits. Weak when the project needs cross-tool memory or a safety model that outlives the editor.

## Agent HQ OS

Adds a project-level operating layer: boot files, memory, roles, validation, and telemetry. It works with ChatGPT, Claude, Claude Code, Codex, Cursor, and local scripts because the core is plain files.
