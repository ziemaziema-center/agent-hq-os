# Multi-Agent Meeting Protocol

## Roles

- HQ Orchestrator frames the task and assigns scope.
- Specialist agents answer bounded questions or implement scoped changes.
- Safety Reviewer checks secrets, live side effects, and irreversible operations.
- Validator verifies behavior before closeout.

## Meeting Shape

1. Context readout.
2. Risk scan.
3. Role assignments.
4. Findings or patch proposals.
5. Decision.
6. Validation.
7. Telemetry update.
