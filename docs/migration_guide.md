# Migration Guide

## From Ad-hoc AI Sessions

1. Add `SESSION_BOOT.md`.
2. Add `memory/KNOWN_FAILURES.md`.
3. Add one validation command or checklist.
4. Add daily execution telemetry.

## From Team Docs

1. Convert recurring review comments into known failures.
2. Convert successful workflows into validated patterns.
3. Add role cards only for roles you actually use.

## From Private Automation Projects

1. Define public/private boundaries.
2. Replace secrets with placeholders.
3. Publish fixtures before live adapters.
