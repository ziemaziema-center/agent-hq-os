# Research Workflow

## User Request

A research task compares provider routing options. HQ separates claims from measurements and stores validated patterns only when evidence exists.

## HQ Plan

- Load memory.
- Identify safety boundaries.
- Assign one specialist role.
- Define validation before edits.

## Agent Delegation

- Specialist produces the smallest complete change.
- Validator checks evidence.
- HQ records the final decision.

## Telemetry

- Success: reusable workflow captured.
- Failure: any blocked assumption becomes a known failure.
