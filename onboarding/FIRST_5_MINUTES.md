# First 5 Minutes

## Minute 1: Copy the Starter Kit

Copy `starter_kit/project_bootstrap/` into a new or existing project.

## Minute 2: Fill Project Boot

Edit `PROJECT_BOOT.md` with the project purpose, safety boundaries, and validation commands.

## Minute 3: Add Known Failures

Add one mistake the project must not repeat to `memory/KNOWN_FAILURES.md`.

## Minute 4: Run a Session

Ask your agent to read `SESSION_BOOT.md` and summarize the validation plan before editing.

## Minute 5: Close the Loop

Record the outcome in `telemetry/DAILY_EXECUTION_LOG.md`.
